# King's Cross Labs Marketplace

This repository hosts Claude marketplace catalogs published by Kings Cross Labs.

The catalog currently points to:

- `instagram-insights` for Instasights from [`nickcruz/instasights`](https://github.com/nickcruz/instasights)

## Install

```text
/plugin marketplace add https://github.com/kingscrosslabs/marketplace.git
/plugin install instagram-insights@kingscrosslabs-marketplace
```
